<h1>Hide user profile fields</h1>

<p>by <a href="http://mariancerny.com/" title="Marian Cerny">Marian Cerny</a></p>

<p>

This plug-in removes unnecessary profile fields from user profiles, namely:
	<ul>
		<li>Personal options (visual editor, keyboard shortcuts, color scheme picker)</li>
		<li>AIM, YIM and Jabber contact fields</li>
		<li>About user (bio)</li>
	</ul>

</p>

<p>
There is no menu, no options. If you want to change which options are hidden, just modify the plugin file.
</p>
